Leçon : Les Tableaux (Array)

Qu'est-ce qu'un Tableau (Array) ?

Un tableau est une structure qui permet de stocker une collection de plusieurs valeurs du même type dans une seule variable. Imaginez une boîte de rangement avec des compartiments numérotés.

Chaque élément du tableau est accessible via un index (sa position), qui commence toujours à 0.

Déclaration et Initialisation

Vous pouvez déclarer un tableau et le remplir en même temps. La taille est alors calculée automatiquement.

// Un tableau d'entiers (int) nommé 'pins'
// Index :   0  1  2  3
int pins[] = {2, 4, 7, 8};

Accéder à un élément

On utilise le nom du tableau suivi de l'index entre crochets [ ].

int premierePin = pins[0]; // Vaut 2
int troisiemePin = pins[2]; // Vaut 7

// On peut aussi l'utiliser directement :
digitalWrite(pins[3], HIGH); // Met la pin 8 à HIGH

Parcourir un tableau avec une boucle for

Les tableaux et les boucles for sont faits pour travailler ensemble. C'est la méthode la plus courante pour manipuler tous les éléments d'un tableau.

// Allumer toutes les LEDs connectées aux pins du tableau
for (int i = 0; i < 4; i++) {
  // Au premier tour, i=0, on utilise pins[0]
  // Au second tour, i=1, on utilise pins[1]... etc.
  pinMode(pins[i], OUTPUT);
  digitalWrite(pins[i], HIGH);
}

Le tableau suivant est défini dans le code. Entrez un index pour lire la valeur qui s'y trouve.

Piège 1 : L'index commence à 0 !

C'est la source d'erreur la plus fréquente. Le premier élément est à l'index [0], le deuxième à l'index [1], etc.

int monTableau[3] = {10, 20, 30};
// Le premier élément est monTableau[0]  (vaut 10)
// Le dernier élément est monTableau[2] (vaut 30)

Piège 2 : L'accès "hors limites" (Out of Bounds)

Si un tableau a une taille de 5, ses index valides sont 0, 1, 2, 3, 4. Essayer de lire ou d'écrire à monTableau[5] est une grave erreur. Arduino ne vous préviendra pas, mais cela corrompra la mémoire et créera des bugs très difficiles à trouver.

int monTableau[3] = {10, 20, 30};
// ERREUR ! L'index 3 n'existe pas.
// Comportement imprévisible du programme !
int valeur = monTableau[3];

Piège 3 : Comment connaître la taille d'un tableau ?

En C++, il n'y a pas de fonction simple comme tableau.length. On utilise une astuce avec l'opérateur sizeof() qui retourne la taille en octets.

int notes[] = {12, 17, 8, 14, 11};
// sizeof(notes) -> 5 * 2 = 10 (un int occupe 2 octets)
// sizeof(notes[0]) -> 2 (la taille d'un seul int)
int taille = sizeof(notes) / sizeof(notes[0]); // 10 / 2 = 5
// 'taille' vaut maintenant 5, on peut l'utiliser dans une boucle !
for (int i = 0; i < taille; i++) { ... }
